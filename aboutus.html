About project

Jobs.co is an innovative job portal tailored specifically for the tech industry, enabling users to seamlessly search and apply for technology roles. The platform integrates advanced tools for building professional resumes and crafting personalized cover letters, enhancing each candidate's application process. This succinct version focuses on the core functionalities and benefits, suitable for promotional materials or a website introduction.


Key Features

  • Generate Personalised Cover Letters with AI
  • Generate ATS Friendly Resume
  • Apply to Jobs
  • Comprehensive Profile Analytical Dashboard

Below is the Data Model in ERD form.



For this project, I utilized HTML, CSS, and JavaScript for the front-end design and user interface, creating a seamless and interactive experience. The back-end functionality, including API calls, was implemented using .NET MVC to ensure robust data handling and responsive service interactions.
Development was supported by tools such as VS Code and Visual Studio, which facilitated an efficient coding environment. The resume builder feature leverages LaTeX to PDF conversion APIs combined with JavaScript to produce professional PDF resumes. Additionally, the cover letter generator employs OpenAI APIs to create personalized content tailored to specific companies and job descriptions. JavaScript is then used to convert this content into formatted PDF documents.
